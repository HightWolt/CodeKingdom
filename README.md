# 🌲 CodeKingdom

Текстовая RPG-игра на Python. Исследуй мир, сражайся с волками, собирай артефакты и выживай!

## 🚀 Как запустить
1. Убедись, что установлен [Python 3.8+](https://www.python.org/)
2. Скачай репозиторий или склонируй его:
   ```bash
   git clone https://github.com/HightWolt/CodeKingdom.git
   cd CodeKingdom
3. Запусти игру:
   `python main.py`
   
## 🎮 Управление
   1. → Пойти в лес (бой, лут)
   2. → Зайти в деревню (лекарство)
   3. → Отдохнуть у костра (+здоровье)
   4. → Выйти из игры
   5. 
## 💾 Сохранение прогресса
   - Игра **автоматически сохраняет** состояние при выходе (опция `4`) или гибели героя.
   - Файл `save.json` создаётся в папке с игрой. Не удаляй его, если хочешь продолжить приключение!
   - При повторном запуске игра предложит: `1. Новая игра | 2. Продолжить`.
   - 
## 📦 Запуск без установки Python
   1. Перейди в раздел **[Releases](https://github.com/ТВОЙ_НИК/CodeKingdom/releases)**
   2. Скачай `CodeKingdom.exe` из последнего релиза
   3. Запусти двойным кликом (Windows)

   **⚠️ Антивирус может временно заблокировать файл. Это ложное срабатывание для самописных `.exe`. Добавь в исключения или скачай исходный код и запусти через `python main.py`.**
   
## 🌙 Живой мир (v3.0)
- **Цикл дня и ночи**: каждые 5 ходов мир меняет фазу. Ночью враги опаснее, днём герой восстанавливает силы.
- **Случайные события**: 35% шанс встречи с аномалиями, находками или испытаниями при каждом выборе.
- Прогресс времени и суток сохраняется вместе с персонажем.

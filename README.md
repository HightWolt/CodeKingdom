# 🌲 CodeKingdom

Текстовая RPG-игра на Python. Исследуй мир, сражайся с волками, собирай артефакты и выживай!

## 🚀 Как запустить
1. Убедись, что установлен [Python 3.8+](https://www.python.org/)
2. Скачай репозиторий или склонируй его:
   ```bash
   git clone https://github.com/ТВОЙ_НИК/CodeKingdom.git
   cd CodeKingdom
3. Запусти игру:
    `python main.py`
## 🎮 Управление
    1 → Пойти в лес (бой, лут)
    2 → Зайти в деревню (лекарство)
    3 → Отдохнуть у костра (+здоровье)
    4 → Выйти из игры
## 💾 Сохранение прогресса
    - Игра **автоматически сохраняет** состояние при выходе (опция `4`) или гибели героя.
    - Файл `save.json` создаётся в папке с игрой. Не удаляй его, если хочешь продолжить приключение!
    - При повторном запуске игра предложит: `1. Новая игра | 2. Продолжить`.
